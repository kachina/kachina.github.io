Cloud Automator 以外の Cloud Automator

Cloud Automator AdventCalendar 2017

Posted by @kachina_t on Wed, Dec 13, 2017
In Development
Tags cloud automator, 2017, adventcalendar

この記事は「Cloud Automator Advent Calendar 2017」の 13 日目の エントリーとなります。

はじめに

全25回の Advent Calendar のうち、ちょうど”まんなか”の13 日目を頂きました。 今回は Cloud Automator をつかってｘｘｘを実現する。ってアプローチからは少し離れて、Cloud Automator を使っていくうえで知っていると便利なサイトを紹介してみたいと思います。

この記事を読んで『こんな使い方があるんだ！』なんて発見に繋がればウレシイです。

1. Twitter

Cloud Automator の Twitter アカウントは @CloudAutomator です。

最新のリリース情報から、雑多な話題まで幅広く発信しているのでフォローしておくと便利ですね。

2. Facebook

Cloud Automator の Facebook Pageです。

動画による操作方法なんかは要チェックですね。

3. マニュアル

操作方法や Tips を解りやすく説明しているのが マニュアル ページです。

Cloud Automator を 日々のバックアップ自動化、自動起動/停止でしか利用していないのであれば必読です！ 構成レビュー、ジョブ連携等、AWS をもっと便利に運用する為の機能が豊富に用意されています。 コスト最適化以上の価値を享受して頂けると思います。

4. サポートページ

サポートへのお問合せ、機能リクエストは こちら です。

です。

投稿されたリクエストは アイデア として公開されます。 このアイデアに対して、他ユーザーが1〜3ptの投票をすることが可能です。 開発チームは、投票数の多いアイデアを積極的に実装しているようなので『これは欲しい！』なんて思うアイデアに積極的に投票することで、Cloud Automator を成長させていくことが可能です。

ちなみに、サポートへの問い合わせはコンソール右下の『？』からも実施することができます。（便利ですね

5. ロードマップ

『つぎはどんな機能がリリースされるの？』なんて思ったら ロードマップ を確認しましょう。 自分が投稿したアイデアがロードマップに入っていたら胸熱です。

6. サービスステータス

メンテナンスの予定や、サービスのステータスを確認したいときた ステータスページ を確認します。

右上の SUBSCRIBE TO UPDATES に登録しておくと、メンテナンスの連絡が届くようになりますのでオススメです。

8. API ドキュメント

『もっとヘビーに Cloud Automator を使い倒したい』って方には API ドキュメント をご紹介します。

もう、こうなったらなんでもできちゃいますｗ

7. 開発ブログ

さいごに、Cloud Automator ってどんな人達が作っているんだろう？なんて思ったら 開発Blog をみてください。

まとめ

バックアップ・自動停止/起動以外にも多くの Tips があることを発見して頂けたと思います。 AWS をもっと便利にするサービス、Cloud Automator ってすばらしいですね！
